# CalcCap 🧢

Sometimes math, sometimes cap.

CalcCap is a playful JavaScript calculator that intentionally gives incorrect answers sometimes.  
It’s a humorous experiment showing that even machines shouldn’t always be trusted blindly.

---

## ✨ Features

- Basic calculator operations (+, -, ×, ÷)
- Randomly gives correct or incorrect results
- Funny messages explaining the answer
- Simple and clean UI
- Built using HTML, CSS, and JavaScript

---

## 🎯 How It Works

When the user enters numbers and selects an operation:

1. The calculator calculates the correct result.
2. A random decision is made:
   - 50% chance → shows the correct answer
   - 50% chance → shows a wrong answer
3. A random message is displayed explaining the result.

Example:

Result: 42
Source: vibes.

 or

 Result: 8
 Okay fine… this one is correct.
